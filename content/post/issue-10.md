##### 1. 强行抹掉远程commit log

> 不小心把不该提交的东西提交了, 可以用这招救命, **团队协作慎用**

```shell
git reset --hard <需要回退到的commit tag>
git push --force
```

##### 2. 强制同步远端代码

> 不小心把本地搞乱, 除了`rm and clone again`大法外, 还可以这样

```shell
git fetch origin master
git reset --hard origin/master
git pull -r
```
