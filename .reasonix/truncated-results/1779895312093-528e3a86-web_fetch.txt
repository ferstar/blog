本文主要解决的问题是

开通Telnet

获取超管权限

运营商与华为操作界面互换

更改MAC/SN/EPON/GPON

激活Lxc容器内的OpenWRT

其他

备份是个好习惯，事先务必记录一些资料

光猫的LOID

PPP/ITV/TR069的VLAN ID，

MAC地址和设备SN

宽带账号与密码

需要准备的工具：

1.ONT维修使能工具（用于开启路由器Telnet）
2.华为配置加解密工具（密文解密-用$2解密）
3.电脑安装有Telnet或者直接下载一个telnet.exe 也可

一、开启Telnet #

光猫拔除所有线缆，然后LAN1口插网线直接与电脑连接，通电

电脑上ipv4地址设置为192.168.1.x（小于255且不为1）

电脑上打开ONT维修使能工具，点V5使能（对应天翼网关的版本），选择连接LAN1口的网卡，按“启动”按钮，等待使能成功（直到当前成功总数为1），此时右侧设备列表会显示绿色的success，光猫上所有指示灯常亮，不用管它，点击停止，拔线重启。

重启完毕后电脑上

<code class="language-shell" data-lang="shell"><span class="line"><span class="cl">telnet 192.168.1.1
</span></span><span class="line"><span class="cl">root <span class="c1">#用户名</span>
</span></span><span class="line"><span class="cl">adminHW <span class="c1">#密码</span>
</span></span><span class="line"><span class="cl">su <span class="c1">#提权，此时只能使用华为内置的定制命令，输入?显示所有可用的命令。</span>
</span></span><span class="line"><span class="cl">shell <span class="c1">#进入shell</span>
</span></span><span class="line"><span class="cl">ls <span class="c1">#能列出文件即表示shell补全成功。</span>
</span></span></code>

此时shell用户其实是srv_ssmp，可以再输入一次su(密码:admin)提升为root。

二、获取超管权限 #

获取超管权限有三种方式

提升useradmin为超级权限用户

查询超管用户telecomadmin密码

增加一个超管用户

先Telnet进去

<code class="language-shell" data-lang="shell"><span class="line"><span class="cl">telnet 192.168.1.1
</span></span><span class="line"><span class="cl">root
</span></span><span class="line"><span class="cl">adminHW
</span></span><span class="line"><span class="cl">su
</span></span><span class="line"><span class="cl">shell
</span></span><span class="line"><span class="cl">
</span></span><span class="line"><span class="cl"><span class="nb">cd</span> /mnt/jffs2/ <span class="c1">#进入/mnt/jffs2目录</span>
</span></span><span class="line"><span class="cl">cp -f hw_ctree.xml hw_ctree.xml.bak <span class="c1">#备份是个好习惯</span>
</span></span><span class="line"><span class="cl">cp -f hw_ctree.xml mycfg.xml.gz <span class="c1">#拷贝一份出来</span>
</span></span><span class="line"><span class="cl">aescrypt2 <span class="m">1</span> mycfg.xml.gz tem <span class="c1">#1为解密 0为加密，tem是key，加密也用这个</span>
</span></span><span class="line"><span class="cl">gzip -d mycfg.xml.gz <span class="c1">#解压</span>
</span></span></code>

到这里为止我们得到了一个解密后的mycfg.xml文件

看看其中用户设置部分

</span></span><span class="line"><span class="cl">
</span></span><span class="line"><span class="cl">----------------------------------------
</span></span><span class="line"><span class="cl">
</span></span><span class="line"><span class="cl">NumberOfInstances=2 用户总数为2
</span></span><span class="line"><span class="cl">InstanceID="1" 用户序号
</span></span><span class="line"><span class="cl">UserLevel="1" 1为普通用户 0为超级用户
</span></span><span class="line"><span class="cl">Enable="1" 1启用 0禁用
</span></span><span class="line"><span class="cl">Alias="cpe-1" 别名，不重要
</span></span><span class="line"><span class="cl">Salt="c3d06c58fc5ad62c23979eff" 盐，加密用的随机数
</span></span><span class="line"><span class="cl">PassMode="3" 加密模式 ，0表示明文不加密
</span></span></code>

搞懂这段代码的含义就可以修改了，根据自己需要选择下面三个操作中的任意一个或几个

1、提升useradmin为超级权限
把useradmin这一行的UserLevel=”1″改为UserLevel=”0″即可

2、查询超管telecomadmin的密码
telecomadmin这一行内$与$之间的xxxxxxxxxxxxxx$就是超管密码，用华为解密工具的密文解密里的$2解密即可

<code class="language-shell" data-lang="shell"><span class="line"><span class="cl">grep WebUserInfoInstance mycfg.xml <span class="c1">#提取包含有超管密码的项password字段</span>
</span></span></code>

另：其实获取超管密码还有另一个相对简单的办法

<code class="language-shell" data-lang="shell"><span class="line"><span class="cl">telnet 192.168.1.1
</span></span><span class="line"><span class="cl">root
</span></span><span class="line"><span class="cl">adminHW
</span></span><span class="line"><span class="cl">display current-configuration grep telecomadmin
</span></span><span class="line"><span class="cl">
</span></span><span class="line"><span class="cl">屏幕上会直接打印出含有telecomadmin账户密码的行，解密即可
</span></span></code>

3.增加一个超管用户

<code class="language-xml" data-lang="xml"><span class="line"><span class="cl">修改
</span></span><span class="line"><span class="cl"><span class="nt"><X_HW_WebUserInfo</span> <span class="na">NumberOfInstances=</span><span class="s">"2"</span><span class="nt">></span>
</span></span><span class="line"><span class="cl">为
</span></span><span class="line"><span class="cl"><span class="nt"><X_HW_WebUserInfo</span> <span class="na">NumberOfInstances=</span><span class="s">"3"</span><span class="nt">></span>
</span></span><span class="line"><span class="cl">
</span></span><span class="line"><span class="cl">在telecomadmin这一行下面新增一行,表示新增一个超管admin密码为newpasswd
</span></span><span class="line"><span class="cl"><span class="nt"><X_HW_WebUserInfoInstance</span> <span class="na">InstanceID=</span><span class="s">"3"</span> <span class="na">ModifyPasswordFlag=</span><span class="s">"0"</span> <span class="na">UserName=</span><span class="s">"admin"</span> <span class="na">Password=</span><span class="s">"newpasswd"</span> <span class="na">UserLevel=</span><span class="s">"0"</span> <span class="na">Enable=</span><span class="s">"1"</span> <span class="na">Alias=</span><span class="s">"cpe-3"</span> <span class="na">PassMode=</span><span class="s">"0"</span> <span class="nt">/></span>
</span></span></code>

完成修改以后

<code class="language-shell" data-lang="shell"><span class="line"><span class="cl">gzip mycfg.xml <span class="c1">#压缩</span>
</span></span><span class="line"><span class="cl">aescrypt2 <span class="m">0</span> mycfg.xml.gz tem <span class="c1">#加密</span>
</span></span><span class="line"><span class="cl">cp -f mycfg.xml.gz /mnt/jffs2/hw_ctree.xml <span class="c1">#替换</span>
</span></span><span class="line"><span class="cl">
</span></span><span class="line"><span class="cl">reboot <span class="c1">#重启生效</span>
</span></span></code>

注意：以上三种方式中，useradmin提权最简便，但如果在前台WEB界面修改useradmin的密码，useradmin可能会变回普通用户，未验证！第二种方式不需要修改文件，但如果运营商下发数据的话应该会发生改变，我采用的是第三种方式新增一个超管用户。

三、运营商界面与华为操作界面互换 #

<code class="language-shell" data-lang="shell"><span class="line"><span class="cl">cp -f /mnt/jffs2/hw_ctree.xml /mnt/jffs2/hw_ctree_bak.xml （备份运营商hw_ctree文件）
</span></span><span class="line"><span class="cl">cp -f /mnt/jffs2/hw_boardinfo /mnt/jffs2/hw_boardinfo_bak （ 备份运营商hw_boardinfo文件）
</span></span><span class="line"><span class="cl">
</span></span><span class="line"><span class="cl">
</span></span><span class="line"><span class="cl">cp -f /etc/wap/hw_default_ctree.xml /mnt/jffs2/hw_ctree.xml （ 拷贝华为原厂hw_ctree文件）
</span></span><span class="line"><span class="cl">
</span></span><span class="line"><span class="cl">sed -i <span class="s1">'s/^.*obj.id = "0x0000001a" ; obj.value =.*$/obj.id = "0x0000001a" ; obj.value = "COMMON";/'</span> /mnt/jffs2/hw_boardinfo
</span></span><span class="line"><span class="cl">sed -i <span class="s1">'s/^.*obj.id = "0x0000001b" ; obj.value =.*$/obj.id = "0x0000001b" ; obj.value = "COMMON";/'</span> /mnt/jffs2/hw_boardinfo
</span></span><span class="line"><span class="cl">sed -i <span class="s1">'s/^.*obj.id = "0x00000031" ; obj.value =.*$/obj.id = "0x00000031" ; obj.value = "NOCHOOSE";/'</span> /mnt/jffs2/hw_boardinfo
</span></span><span class="line"><span class="cl">
</span></span><span class="line"><span class="cl">reboot （重新启动）
</span></span></code>

1a, 1b, 31这三行sed替换的命令 用vi直接编辑hw_boardinfo其实更加方便。

注意：因为替换了华为原厂的ctree文件，此时连接telnet的话，需要把电脑的IP地址修改为192.168.100.0/24地址段的，连接192.168.100.1 ，WEB登陆的密码则是telecomadmin: admintelecom，登录进去以后恢复设置就可以以192.168.1.1来连接了。

恢复电信界面的话反向操作即可，要把上面那三行替换为COMMON和NOCHOOSE的项再换回去
以湖北电信为例

<code class="language-shell" data-lang="shell"><span class="line"><span class="cl">telnet 192.168.100.1

</span></span><span class="line"><span class="cl">root
</span></span><span class="line"><span class="cl">admin

</span></span><span class="line"><span class="cl">
</span></span><span class="line"><span class="cl">sed -i <span class="s1">'s/^.*obj.id = "0x0000001a" ; obj.value =.*$/obj.id = "0x0000001a" ; obj.value = "E8C";/'</span> /mnt/jffs2/hw_boardinfo
</span></span><span class="line"><span class="cl">sed -i <span class="s1">'s/^.*obj.id = "0x0000001b" ; obj.value =.*$/obj.id = "0x0000001b" ; obj.value = "HUBCT";/'</span> /mnt/jffs2/hw_boardinfo
</span></span><span class="line"><span class="cl">sed -i <span class="s1">'s/^.*obj.id = "0x00000031" ; obj.value =.*$/obj.id = "0x00000031" ; obj.value = "CHOOSE_XINAN";/'</span> /mnt/jffs2/hw_boardinfo
</span></span><span class="line"><span class="cl">reboot 
</span></span></code>

如果要永久启用华为界面，会清除运营商的定制信息，那可就再也回不去运营商界面了，慎用!

<code class="language-shell" data-lang="shell"><span class="line"><span class="cl">restorehwmode.sh

替换为华为界面的话会多出来很多设置，但对家庭用户而言大部分没什么用，比较有价值的是可以设定dhcp静态绑定和删除tr069，但是无法开放usb的匿名共享。

说到这个tr069的话，在运营商界面里是没法删除和禁用的，在华为的界面里倒是可以直接操作，但是telnet显然更方便一些

<code class="language-sh" data-lang="sh"><span class="line"><span class="cl">telnet 192.168.1.1

</span></span><span class="line"><span class="cl">ip interface config interface wan1 status Disable <span class="c1">#禁用tr069 Enable启用</span>
</span></span></code>
